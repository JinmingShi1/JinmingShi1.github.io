Bilingual Subtitles Chrome Extension

browser extension · real-time translation

A Chrome extension (Manifest V3) that overlays original + translated subtitles on streaming sites. The core: intercepting fetch/XHR in the page's main world to capture subtitle files as the player loads them, parsing multiple formats (VTT / TTML / native text tracks as fallback), batch-translating lines via LLM API with line-number mapping, and rendering an overlay synced precisely to the video timeline. Bring-your-own-API-key design — user data never touches a third-party server.

intercept subtitle requests→multi-format parsing→batched LLM translation→timeline-synced overlay
Chrome Extension (MV3) JavaScript fetch/XHR interception VTT / TTML parsing DeepSeek API
🔒 Personal use — open-sourcing on request
